## Copilot Assisted Coding
**GitHub Copilot** is basically an AI assistant that helps you write code faster. It fits right into **Visual Studio Code** and can help to generate repetitive code, write tests, or explain confusing parts of a file if you ask it in plain English. Just remember that it's not perfect—you still need to verify what it writes, especially for complex logic. Think of **Copilot** as a smart learning buddy to enhance your coding skills rather than a replacement for your own coding skills.

## Data Science Guide
Data science is all about making sense of messy data using a mix of coding, mathematics, statistics, domain knowledge, and business knowledge. Most of the work involves gathering data, cleaning it up, exploring it through data visualization, analysing it,and building machine learning models that can make predictions. It's how companies make smarter decisions, like predicting what customers will buy or improving healthcare. To do this well, it requires technical skills in languages like **Python** or **R**, plus the ability to communicate your findings clearly to non-technical people.

## Git Version Control
**Git** is a tool that tracks every change you make to your code so you never lose your work. It thinks about your project in three stages: your working folder, a staging area for things you're about to save, and the repository where everything is permanently saved. By the use of commands like `add`, `commit`, and `push`, you can save your progress safely and try out new ideas without breaking your main project. When combined with platforms like **GitHub**, it makes working with other developers super easy.

## Marimo Notebooks
**Marimo** is a reactive Python notebooks with hot reload feature. it saves everything as normal **Python** files so it plays perfectly with version control. The best part is that it's reactive i.e if you change code in one cell, any other cell that depends on it updates automatically. **Marimo** has full python support and uses `.py` extension which is beautiful.

## Markdown Writing
**Markdown** is a simple way of formatting plain text without the need of editor like Ms Word. You just use basic symbols—like asterisks to make things bold or hashtags for headers. Because the files are just plain text, you can open them anywhere, and they look great when published on places like **GitHub**, **Slack**, or **Jupyter**. It is a common way to write documentation with ease and no stress of designing texts and layouts.

## Plotly Visualization
**Plotly** is a library that takes in some data and represent them as nice, interactive charts you can click around and explore. If you want something quick, **Plotly Express** lets you make things like scatter plots or bar charts in just one line of code. If you need something very specific with customization, you can dive deeper into its **Graph Objects**.

## Polars Data Wrangling
**Polars** is a dataframe library for python for data manipulation, written in Rust. It's fast and more memory-efficient than **Pandas** because of its parallel processing capabilities. Other beautiful features of **Polars** includes type safety and its moderate learning curve.

## Python Guide
**Python** is a popular programming language that's great for data science and analysis, but it's also used for web development, automation, and more. It's usually recommended for beginners because it's easy to learn because of its large community. It's designed to be highly readable since it uses spacing and indentation instead of curly braces. This guide covers the absolute basics so you can get started quickly: variables, standard data structures (like lists and dictionaries), and loops. It also shows you how to write functions and read error messages when your code breaks.

## UV Package Management
**UV** is a modern tool package manager for python. It's built in Rust and serves as a better alternative to older Python tools like pip because of its speed (10-100x faster than pip), reliability and simplicity. It is fast at downloading and installing packages because it does everything in parallel and caches what it downloads. With basic commands like `uv init`, `uv add`, and `uv sync`, you can set up a project and install the dependencies you need in seconds. Best of all, it handles your virtual environments and **Python** versions for you automatically, so you don't have to think about them.

## VS Code Guide
**Visual Studio Code** (or **VS Code**), microsoft-owned free code editor, is a popular code editor with super features. It gives you features like code autocompletion (IntelliSense), syntax highlighting, and a terminal right at the bottom of the screen. The guide teaches how to rely on keyboard shortcuts and the Command Palette so you can work and navigate much faster. **VS Code** also has a massive library of extensions and built-in **Git** tools, making it easy and best to work with.
